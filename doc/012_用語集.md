## 用語集

# LM（ランドマーク）

顔の各点の位置情報

# Region（リージョン）

コントローラのアトリビュートがupper / lower / eye / eyelidのどれに該当するかの情報

upper：眉周りの動き

lower：鼻、頬、口周りの動き

eye：UIではgaze表記、目線の動き

eyelid：まぶたの動き

# Retarget（リターゲット）

ピックアップしたフレームの表情を登録すること、またはリターゲットしたデータのこと

# Solve(ソルブ)

リターゲットしたデータを計算してアニメーションに出力すること

FCSではmayaシーンに音声のセット、イメージプレーンへの画像セットも行う
