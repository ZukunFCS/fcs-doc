Development roadmap

Update February 27, 2026

FCS is currently actively promoting development to add new features and continuously improve quality, and as part of this ongoing effort, we are releasing part of our development map road below.

This roadmap is presented as a reference for current development plans and does not guarantee the implementation of specific features in future software updates.
Additionally, the order of the listed items does not necessarily indicate the priority of implementation, and the content may be changed or discontinued without notice depending on the development status or other circumstances.
(As a result, we are also listing items that have been changed or canceled from previous plans.)
Thank you for your understanding.

Main items planned for future implementation

  • Improved CLI (Command Line Interface) functionality
    We plan to provide an unattended mode that allows video processing to be performed without using a GUI.
    It can be operated from the command line and can be integrated with batch processing, helping to automate and streamline a series of workflows.


  • Post processing function
    We are currently developing several post-processing methods to automatically exaggerate and adjust performance expressions more effectively.


Continuously promoting functional enhancements and improvements

  • Improvements to the landmark detection function
    FCS’s proprietary facial expression detection engine will continue to be a top priority.
    In particular, we will continue to improve detection accuracy so that we can accurately capture even subtler changes in facial expressions in close-up images captured with a head-mounted camera (HMC).


  • Faster processing
    We will optimize the entire software to shorten the processing time from analysis to output.
    We will continue to improve the internal structure to provide a more efficient workflow in production environments where large-scale data processing and short delivery times are required


Items for which development was abandoned or discontinued due to various circumstances

  • Real-time video generation Create a program to analyze and output animations in real time
    We have decided not to include this function as a standard feature in FCS.
    Going forward, we will place emphasis on optimizing for individual project requirements and consider providing individual support. Please contact us for consultations regarding individual support.


  • Development of sample engine code (Unity, Unreal Engine, Motion Builder).


  • Equipped with automatic retargeting functions An algorithm adds the capability to automatically perform rough retargeting on any rig


  • Retargeting support with Unreal Engine Extending the Maya features in FCS to Unreal Engine


FCS aims to be a tool that can make a deeper contribution to our customers’ actual production sites, so we are always open to receiving requests for additional features.In addition to providing standard functions, we can also customize builds optimized for your company’s specific production pipeline and workflow[1]If you have any questions or requests about our products, or any technical issues you may be facing during implementation, please feel free to contact us via the [HELTEC website].(https://sensing.heltec.co.jp/product/zukun-lab/fcs/)
