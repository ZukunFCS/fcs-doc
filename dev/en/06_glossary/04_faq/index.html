Frequently Asked Questions Q&A

About Using FCS

Q: Is an internet connection required when operating FCS?

An internet connection is required. The internet is needed for periodic license information checks.
Wi-Fi connections are supported, but a wired connection is recommended.

Q: Are there any PC specifications requirements?

PCの推奨スペックなどFCSを使用するにあたって必要な準備については、
事前準備のページをご確認ください。

Q: Is a dedicated head-mounted camera required?

専用のカメラは不要です。
市販のヘッドマウントカメラや固定カメラなどの動画もご使用いただけます。

詳しくは事前準備のページをご確認ください。

Q: Is macOS supported?

現時点では対応していませんが、将来的には対応する予定です。※2024年6月現在

※2026年4月更新
MasOSには対応しておらず、対応の予定もございません。
Linuxなどへの対応についてはお問い合わせください。

Q: Is any other software required to use FCS?

Maya is required. (Compatible with Maya 2018/2020/2022/2023/2024/2025/2026)


About FCS Features

Q: Does FCS require markers to be placed on the actor during filming?

Since FCS features auto-tracking, there is no need to place markers on the actor.

Q: Can landmarks be corrected when they are misaligned?

Because auto-tracking is used, landmark misalignment cannot be manually corrected.

Q: Is tongue tracking supported?

Tongue tracking is not supported.

Q: Does it support skeletons in addition to morphs?

FCS supports not only morphs but also skeletons.

Q: The same actor plays multiple characters. Can the Profile be reused?

Yes, if the rig is the same.
Create a new session file using Export and change the linked Maya file.

ExportについてはSessionのエクスポート をご確認ください。

Q: When creating a session for a new character, can session data be reused between the same actors, or can controller information from the same character be used when creating a session for a different actor?

Sessionのエクスポートが利用できます。

登録コントローラーの一括リネーム機能もございますので、適宜コントローラー名の変更を行ってください。 一括リネームについては、ユーザーガイド/Menu/Window/controllersの『複数コントローラーの一括リネーム』の項目をご確認ください。

Q: Can ROM exercise profiles be reused between different actors?

The current FCS algorithm is designed with the assumption that the actor is the same person.
While it is technically possible, reusing profiles between different actors is not recommended from an output accuracy perspective.

Q: Is it necessary to launch Maya separately?

FCSでの作業中は基本的にはMayaの起動が必要になります。
プロジェクトファイルを一度設定すれば、FCSからMayaをワンクリックで起動できます。 Mayaの起動をご確認ください。

FCSCLIをご使用の場合は、こちらをご確認ください。
※外部サイトへ移行します。

Q: Are ROM exercises required for a Profile?

ROM exercises are not required for a Profile, but adding them is recommended to improve accuracy.

Q: Due to plugins, I need to launch Maya separately. What should I do?

In that case, please manually open the port in Maya.
After launching Maya, enter the following command in the Script Editor (MEL):
commandPort -stp "python" -n ":42069" If your technical staff adds the same command to the Maya startup script, the same behavior can be achieved.
If port 42069 is already in use, you can change the port in Settings.

Q: Can controller rigs other than those in the manual be added?

Any controller rig can be added as long as it can be classified into Upper/eyelid/gaze/lower, even if it is not listed in the manual.

Q: Are controller rigs other than Metahuman supported?

Any rig with controllers is basically supported.
Controller registration and animation output are also possible with blend shapes and joints.

Q: Regarding analysis videos, is it fine if the shooting location changes as long as the field of view is the same?

There is no problem if the shooting location changes as long as the field of view is the same.
This can be handled by adding pick-ups to the Profile.

Q: Does video quality (Full HD vs 4K, etc.) affect auto-tracking accuracy? Are there other impacts? Any recommendations?

Except for extremely low resolution (480P or below) or poor quality footage (very dark lighting), video quality has little impact on auto-tracking accuracy.

Q: Does video refresh rate (60fps vs 120fps, etc.) affect auto-tracking accuracy? Are there other impacts? Any recommendations?

Please shoot in well-lit environments if possible. For action scenes, using a camera with a higher frame rate to minimize motion blur should be sufficient.

Q: Are there cases where a restart is required?

Various settings can be changed in FCS under File→Setting.
Changes take effect after a restart, so a restart is required.

Settingについては環境設定（File ▶ Settingsウィンドウ）をご確認ください。

Q: Can layouts be saved?

FCSの View→Layout から現在使用しているレイアウトの情報を保存できます。

レイアウトの保存や変更に関して詳しくはユーザーガイド/View/Layoutの項目をご確認ください。

Q: What should I do when FCS is extremely slow?

Turning on Maya Sync when Maya is not running will cause slow performance. If Maya Sync is ON, please turn it OFF.

Q: What is the difference between “A-I-U-E-O” and “Pa-Pi-Pu-Pe-Po” in the ROM exercise materials? Are both necessary?

Referring to pages 46-47 of “zukun_facialROM.pdf” included in the starter kit, select 5 open mouth shapes for “A-I-U-E-O” and 1 lip tuck (plosive) shape for “Pa-Pi-Pu-Pe-Po” when registering the profile.

Q: When registering a profile in the Editor, can values be entered directly into the slider?

You can enter values directly by Ctrl+clicking.

Q: Why does the image plane go behind the model when animation is output?

FCSではMaya内で使われるカメラの指定が可能です。指定した名前のカメラがシーン内に無い場合は新しく作成されます。

「Settings/Maya/Image Plane」や「Settings/Maya/camera」から、使用するカメラやイメージプレーンの名前を設定してください。

これらの設定については下記をご参照ください。

ライセンスについて

Q: With one license, can I work on two PCs simultaneously?

No. However, you can work on multiple PCs as long as FCS is not running on them simultaneously.

Q: How much does it cost?

年間サブスクリプションは40万円（税抜き）です。

Q: Is the contract automatically renewed after purchase?

The contract is not automatically renewed after purchase. A popup will appear in the FCS software one month before the subscription ends, and you will need to contact our representative if you wish to renew.

Q: What should I do if I want to stop renewing?

No special action is needed. You can use FCS until the subscription period ends, but it will become unavailable after that.

Q: Is there a refund if I want to stop using it after 6 months?

Since it is an annual subscription contract, there are no refunds for early cancellation.

Q: How long can I use the trial?

The trial is available for 30 days. For details, please contact us via the HELTEC website.

Q: What domains need to be accessible to use FCS?

The required domains are: api.toei-fcs.com, google.com

Q: Is the contract automatically renewed after the trial?

A separate purchase procedure is required to transition from the trial version to the full version. For details, please contact us via the HELTEC website.

Q: I see an “Internal Server Error” message and cannot use FCS. What should I do?

There is an issue with the server that checks license information.
Server-side action is required, so please contact us via the HELTEC website.
