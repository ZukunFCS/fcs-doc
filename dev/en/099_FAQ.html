Frequently Asked Question

Using FCS

Q: Is an Internet connection required when operating the FCS?

FCS requires constant internet access for checking the validity of the license. WiFi should work but we recommend using wired internet.

Q: Are there any specific requirements for PC specifications?

Minimum Specs:
CPU: Intel Core i5-9400F
GPU: -
Memory: DDR4 16GB or more
Recommended Specs:
CPU: Intel Core i7-13700K
GPU: Nvidia RTX 3080
Memory: DDR4 128GB or more

Q: Is a dedicated head-mounted camera required?

No specific camera is required. Commercial cameras (i.e., Gopro, iPhone) would work, both table mounted or head mounted. You can test your own footage with our 30 days free trials. (No credit card required.)

Q: Is MacOS supported?

It is not supported at this time, but it is planned to be supported in the future. (As of June 2024)

Q: Is any additional software required to use the FCS?

Maya is required. (Compatible with Maya 2018/2020/2022/2023/2024)


FCS features

Q: Does FCS require markers on the actor footage?

Since FCS is equipped with auto-tracking, there is no need to apply markers to the actor.

Q: Can the landmarks be corrected if they are misaligned?

Due to auto-tracking, landmark misalignment cannot be corrected.

Q: Is tongue tracking possible?

Tongue tracking is not supported.

Q: Does it support skeletons as well as morphs?

FCS supports not only morphs but also skeletons.

Q: The same actor is playing multiple characters. Is it possible to reuse their profile?

It is possible if it’s the same rig. Please create a new session file through Export, and change the Maya file you want to link.

Q: When creating a session for a new character, can session data be reused between the same actors, or can controller information for the same character be used for sessions with different actors?

The Export function can be used. A batch rename feature for registered controllers is also available, so please change the controller names as needed.

Q: Is it possible to reuse ROM exercise profiles between different actors?

The current FCS algorithm is designed on the assumption that the actor is the same person. While it is operationally possible, reusing profiles between different actors is not recommended from an output accuracy perspective.

Q: Is it necessary to launch Maya separately?

Once the project file is set up, you can launch Maya with a single click from FCS.

Q: Is live character animation possible?

It is not possible at this time, but we plan to provide support in the future. (As of June 2024)

Q: Is ROM required for the profile?

While ROM is not mandatory for the profile, it is recommended to add it for improved accuracy.

Q: Due to plugin requirements, it is necessary to launch Maya separately. How should I proceed?

In that case, please manually open the port in Maya. After launching Maya, enter the following command in the script editor (Mel): commandPort -stp “python” -n “:42069” If you ask your technical staff to add the same command to the script that launches Maya, you will achieve the same behavior. If port 42069 is already in use, you can change the port in Settings.

Q: Is it possible to add controller rigs other than those in the manual?

If the controller rig other than the instruction manual can be categorized as Upper/eyelid/gaze/lower, it can be added optionally.

Q: Does it support controller rigs other than Metahuman?

It generally supports rigs with controllers. Blend shapes and joints can also be registered as controllers and allow for animation output.

Q: Regarding analysis videos, is it okay if the filming location changes as long as the angle of view remains the same?

As long as the angle of view is the same, it is not a problem if the shooting location changes. You can accommodate this by adding a pickup in the profile.

Q: Does a change in video quality (e.g., Full HD and 4K) affect auto-tracking accuracy? Are there any other areas that may be impacted? Do you have any recommendations?

Except for extremely low resolution (below 480P) or poor quality (extremely dark lighting) videos, the quality of the video has little impact on auto-tracking accuracy.

Q: Does a change in video refresh rate (e.g., 60fps or 120fps) affect auto-tracking accuracy? Are there any other areas that might be impacted? Do you have any recommendations?

If possible, please shoot in a well-lit area. Additionally, for action scenes, it is advisable to use a high frame rate camera to minimize motion blur.

Q: Are there situations where a restart is required?

You can change various items in FCS under File → Settings. Changes will be reflected after restarting, so a restart is required. :::{figure} images/Set001.jpg :width: 80% :align: center :::

Q: Is it possible to save layouts?

You can save the layout information at registration in FCS under View → Layout.

  • Activate the “Layout Registration” window by selecting View → Layout → Save Current.

  • Enter the “Layout Name”

  • You can register by clicking Save
    :::{figure} images/Layout_02_Open-Save.jpg :width: 80% :align: center :::

Q: What should I do when FCS is extremely slow?

If Maya Sync is turned ON while Maya is not running, FCS will become slow. If Maya Sync is ON, please turn it OFF.

Q: What is the difference between “A-I-U-E-O” and “Pa-Pi-Pu-Pe-Po” in the ROM exercise materials? Are both necessary?

Referring to pages 46-47 of “zukun_facialROM.pdf” included in the starter kit, please register profiles by selecting 5 open shapes for “A-I-U-E-O” and 1 lip-tucking (plosive) shape for “Pa-Pi-Pu-Pe-Po”.

Q: When registering profiles in the Editor, is it possible to directly input values into the slider?

You can directly input values by Ctrl+clicking.

Q: Why does the image plane go behind the model when outputting animation?

In FCS, you can specify the camera used in Maya. If a camera with the specified name does not exist in the scene, a new one will be created. Please set the camera and image plane names from “Settings/Maya/Image Plane” and “Settings/Maya/camera”.


Regarding license

Q: With a single license, is it possible to work on two PCs at the same time?

No, it is not possible. However, you can work on multiple PCs as long as they are not launched simultaneously.

Q: How much does the subscription cost?

The annual subscription is 400,000 yen (excluding tax). For those who purchase by the end of December 2024, an additional license will be granted free of charge for each license purchased.

Q: Will the subscription automatically renew after purchase?

After purchase, the contract will not automatically renew. A popup will appear in the FCS software one month before the subscription ends, and if you wish to renew, you will need to contact our representative.

Q: What should I do if I don’t want to renew the subscription?

No special actions are required. You can use it until the subscription period ends, but it will no longer be usable afterward.

Q: If I want to stop using it after 6 months, will there be a refund?

Since it is an annual subscription contract, there are no refunds for early cancellation.

Q: How long can I use the trial version?

You can try the trial for 30 days. For more details, please contact us through the HELTEC webpage.

Q: What domains does FCS connect to?

The domains are: api.toei-fcs.com, google.com

Q: Will the subscription automatically renew after the trial?

A separate purchase process is required to transition from the trial version to the full version. For more details, please contact us through the HELTEC website.

Q: “Internal Server Error” message pops up, and I cannot use the FCS. What should I do?

There is an issue with the server that checks the license information. Since action is required on the server side, please contact us through the HELTEC website.
