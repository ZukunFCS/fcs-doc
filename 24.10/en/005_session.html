Create or open a SessionпғҒ

In FCS, a file that links actor information, character information, Maya scene information, and analysis data is called a вҖңSession.вҖқ

After starting FCS, go to вҖңNewвҖҰ(Create new Session)вҖқ or вҖңOpenвҖқ to access the Session data.

Note

By settings details for a Session at the beginning, you can start working with the buttons on FCS without having to operate Maya separately.

Creating a new SessionпғҒ

  • Fileв–¶Sessionв–¶Select NewвҖҰ

1. Project Folder: Select the location where you want to save the FCS project data
2. Actor: Motion capture actor name
3. Character: 3D model character name
4. Maya Scene: Path to the 3D model Maya scene
5. Maya Base: Path to the folder where Assets and workspace.mel are located
6. Maya Ver: Select the version of the data created in the scene

в‘ Project Folder settingsпғҒ

Click the Browse button to launch a window for selecting the project folder.

  • Select the folder where you want to save FCS data

Create Project Folder.

  • Create

A pop-up will appear after the folder is successfully created.

  • close

Facial and FCS folders will be created.

Note

After creating Project Folder

  • Put Maya scene at Project Folderв–¶Facialв–¶Assets folder

  • It is recommended to move the video you want to analyze to the Project Folder в–¶ Facial в–¶ Recdata folder. *Note: You can still access the video if it is saved in a different location.

в‘Ў Actor settingsпғҒ

Click вҖң+вҖқ and launch вҖңCreate new actor folderвҖқ to create an Actor Folder.

  • Enter the name you want to register in вҖңActor Name.вҖқ

  • вҖңActorвҖқ пјқ name of the motion actor

  • Create

A pop-up will appear after the folder is successfully created.

  • close

The Actor folder you entered will be created directly under the Project Folder folder in Explorer.

в‘ўSetting CharacterпғҒ

Click the вҖң+вҖқ button to launch the вҖңCreate New Character FolderвҖқ window to create a character folder.

  • Enter the desired name in the вҖңCharacter NameвҖқ input field.

  • Create

The Character folder you entered will be created directly under the Actor folder in Explorer.

в‘ЈSetting Maya SceneпғҒ

Click the Browse button to launch a window for selecting a MayaScene.

  • Specify the path for the MayaScene data.

в‘ӨSetting Maya BaseпғҒ

Click the Browse button to launch the window to select the MayaBase.

  • Select the location where workspace.mel is located (the location registered in the Maya scene project settings)

Attention

вҖңworkspace.melвҖқ is not displayed in the popup window in FCS

в‘ҘSetting Maya VerпғҒ

  • Select the version of Maya that was used to create the scene in в‘Ј

After completing all inputs, please press the Save button.

  • Save

fcs_session.yaml (FCS file) will be created directly under the character folder in Explorer.

Note

The .lock file is used to prevent others from accessing the file while you are working on it. They are automatically deleted when you finish your work properly.

Note

If the .lock file remains because of incorrect shut down, etc, please delete the file from the popup when you are starting FCS or delete the file directly in Explorer.


If a Session has already been createdпғҒ

Open Session from history or fcs_session.yaml file.

Open Session from historyпғҒ

If you have started the Session before, the history will be displayed under Fileв–¶Sessionв–¶Open.

  • Click on the session to work on

Open Session from fcs_session.yaml fileпғҒ

  • Fileв–¶Sessionв–¶Openв–¶OpenвҖҰ After you open the Open Session window, local and network drives will be displayed.

  • Select and open the fcs_session.yaml file located directly under the Character folder.


Cautions regarding opening a SessionпғҒ

Startup multiple sessions concurrentlyпғҒ

Warning

While creating/opening a new Session, you cannot create or start another Session. To open another session, close the current session, restart FCS, and reopen it.

If вҖңв‘ҘMaya Ver settingвҖқ is not reflectedпғҒ

You can check the setting details when you create the session under Fileв–¶Sessionв–¶info.

If the MayaVersion setting in NewSession is not reflected in info, right-click MayaVersion on the info screen and select Edit so that you can edit the details.

Attention

Please click the Save button after changing settings.
